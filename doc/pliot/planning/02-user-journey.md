# 02. 사용자 여정 (User Journey)

## 🗺️ 주요 사용자 시나리오

### 시나리오 1: 첫 수업 시작 (신규 사용자)

**목표**: 첫 사용 후 5분 내 수업 시작 완료

**사용자**: 특수체육교사 (중급 기술 수준)

#### 단계별 플로우

| 단계 | 화면 | 행동 | 결과 | 클릭 수 |
|------|------|------|------|---------|
| 1 | 대시보드 | URL 접속 또는 앱 실행 | 메인 화면 표시 | 0 |
| 2 | 대시보드 | 클래스 선택 드롭다운 클릭 | 클래스 목록 표시 | 1 |
| 3 | 대시보드 | 원하는 클래스 선택 | 클래스 선택 완료 | 1 |
| 4 | 대시보드 | "수업 시작하기" 버튼 클릭 | 캘리브레이션 화면 이동 | 1 |
| 5 | 캘리브레이션 | 학생들 배치 지도 | 자동 인식 대기 | 0 |
| 6 | 캘리브레이션 | 학생들이 손 들기 | 자동 매칭 완료 | 0 |
| 7 | 캘리브레이션 | "확인 완료" 버튼 클릭 | 수업 진행 화면 이동 | 1 |

**총 클릭 수**: 4회  
**예상 소요 시간**: 3분  
**검증 결과**: ✅ IA 목표 "5분 내 수업 시작" 충족

#### 예상 이탈 지점 및 대응
- **권한 요청 거부 (15%)**: 명확한 안내 메시지 + 설정 안내
- **캘리브레이션 실패 (10%)**: "다시 시도" 버튼 + 수동 매칭 옵션

### 시나리오 2: 수업 중 모니터링

**목표**: 실시간으로 학생들의 진행 상황 확인 및 제어

**사용자**: 특수체육교사

#### 단계별 플로우

| 단계 | 화면 | 행동 | 결과 | 클릭 수 |
|------|------|------|------|---------|
| 1 | 수업 진행 (교사용) | 6인 그리드 확인 | 실시간 진행률 모니터링 | 0 |
| 2 | 수업 진행 (교사용) | 특정 학생 카드 확인 | 개별 진행 상황 확인 | 0 |
| 3 | 수업 진행 (교사용) | 우측 피드백 로그 확인 | AI 피드백 내역 확인 | 0 |
| 4 | 수업 진행 (교사용) | "일시정지" 버튼 클릭 | 수업 일시정지 | 1 |
| 5 | 수업 진행 (교사용) | "계속하기" 버튼 클릭 | 수업 재개 | 1 |
| 6 | 수업 진행 (교사용) | "다음 운영" 버튼 클릭 | 다음 운영으로 전환 | 1 |

**총 클릭 수**: 1-3회 (제어 시)  
**핵심 정보 접근**: 시각적 즉시 인지 (0클릭)  
**검증 결과**: ✅ 핵심 기능 1-2클릭 내 접근 가능

#### 모니터링 정보 계층
```
[Critical] 즉시 인지 필요
- 현재 운영 이름 및 횟수
- 학생별 완료 여부 (색상/아이콘)
- 전체 진행률

[Supporting] 맥락 이해용
- 개별 학생 카운트
- AI 피드백 로그
- 침궤 상태/FPS

[Optional] 필요 시 확인
- 음량 조절
- 전체화면 토글
```

**목표**: 수업 종료 후 결과 저장 및 새 수업 준비

**사용자**: 특수체육교사

| 단계 | 화면 | 행동 | 결과 | 클릭 수 |
|------|------|------|------|---------|

| 2 | 확인 모달 | "종료" 버튼 클릭 | 결과 화면으로 이동 | 1 |
| 3 | 수업 결과 | 스크롤로 학생별 성과 확인 | 전체 학생 결과 확인 | 0 |

```
- 총 운동 횟수

[Supporting] 맥락 이해용
- 학생별 결과 카드
- 개별 운영 달성률
- AI 코멘트

[Optional] 필요 시 확인
- 저장/공유 버튼
- 상세 통계
```

## 🎭 학생용 화면 여정 (TV View)

### 시나리오: 수업 참여

**목표**: 자신의 캐릭터와 진행 상황을 직관적으로 인지

**사용자**: 지적장애/발달장애 학생 (초보 기술 수준)

#### 시각적 피드백 플로우

```
1. 대기 화면
   → "수업을 기다리고 있어요" 메시지
   → 따라랑 캐릭터 애니메이션

2. 캘리브레이션 화면
   → 6개 캐릭터 슬롯 표시
   → "손을 들어 자리를 찾아보세요" 안내
   → 학생 위치에 따라 캐릭터 활성화

3. 수업 진행 화면
   → 상단: 현재 운영 이름 + 전체 진행률
   → 중앙: 6개 캐릭터 + 각자 이름 + 횟수
   → 하단: AI 피드백 메시지
   → 본인 캐릭터에 효과/애니메이션

4. 운영 완료 시
   → ⭐ 별 애니메이션
   → "잘했어요!" 메시지
   → 박수 소리

5. 수업 종료 화면
   → "수업 끝! 잘했어요!"
   → 전체 달성 현황
   → 축하 애니메이션
```

#### 인터랙션 패턴
- **입력**: 없음 (AI 추적 기반, 터치/클릭 불필요)
- **피드백**: 시각적 (캐릭터 애니메이션) + 청각적 (TTS)
- **시간**: < 500ms 실시간 반응

## 🔄 반복 작업 효율성

### 기존 교사의 다음 수업 시작

**여정**: 대시보드 → [이어하기] 또는 클래스 선택 → [시작]

**최적화**:
1. **마지막 클래스 고정**: 대시보드에 마지막 사용 클래스 상단 표시
2. **빠른 시작**: "다시 시작" 버튼 (설정 유지)
3. **키보드 단축키**: Enter 키로 수업 시작

**예상 클릭 수**: 2회 (이어하기) 또는 3회 (새 클래스)

**검증 결과**: ✅ 반복 작업 효율성 우수
